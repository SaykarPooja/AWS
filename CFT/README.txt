CFT: Cloud Foramtion Template

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
What is CFT?

A CFT is a YAML or JSON file that describes all the AWS resources you want to create.

It is used by AWS CloudFormation to automatically build your infrastructure.

CFT = A single file that tells AWS what to create.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Difference between AWS CFT and CLI?

CFT = Infrastructure as Code

Reusable, automated, predictable, version controlled.

CLI = One-time manual commands

Not reliable, not repeatable, not scalable.
